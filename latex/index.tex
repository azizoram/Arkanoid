It is necessary to develop a computer program that implements the game \char`\"{}\+Arkanoid\char`\"{} in the C++ programming language. This program implements the classic version of the \mbox{\hyperlink{classArkanoid}{Arkanoid}} game. The main window contains a platform that moves horizontally from one wall to another, blocks fixed in place, and a ball that bounces off the platform, walls and blocks, destroying them in the process.
